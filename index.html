Корейский словарь: быстрый поиск + похожие слова

Загружай несколько SQLite-файлов словаря .db. Отдельно можно загрузить nk200sg7_vectors.db, чтобы искать семантически близкие корейские слова. Готово для GitHub Pages: один статический index.html, без сервера. Ничего не отправляется на сервер: файлы обрабатываются и сохраняются в браузере.

Как это использовать

Обычный словарь: загрузи свои dictionary .db и ищи по корейскому, русскому или английскому тексту.

Умный режим: дополнительно загрузи nk200sg7_vectors.db, нажми “Похожие слова” — приложение покажет корейские слова из похожего контекста.

Без зависаний: результаты подгружаются порциями. Можно нажимать “Загрузить ещё”.

GitHub Pages: загрузи этот файл как index.html в репозиторий и включи Pages. Базы не нужно коммитить: пользователь загружает их один раз, дальше они лежат в памяти его браузера.

Память: технически базы сохраняются в IndexedDB. В localStorage хранится только маленькая настройка, включено ли автосохранение.
